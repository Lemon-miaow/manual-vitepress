# 欢迎来到 Vdrias World!

*Vdrias World!* 是一个相对纯净的 Minecraft Java 版生存服务器 ~

::: tip 新周目！

2023年6月，原服主Mashirl重新接手Vdrias World服务器，目前正处于 **第五周目** 的**公开测试**阶段，因此可能会有不稳定的地方，请随时反馈。

:::

- 地址：`play.vdrias.cn`
